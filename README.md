# Ultimate Dungeon

Ultimate Dungeon is a Unity project targeting Unity 6 (URP).

## Getting Started (Unity)
1. Open **Unity Hub**.
2. Click **Add** and select this repository root folder.
3. Open the project with **Unity 6 (URP)**.
4. Once the editor finishes importing, press **Play** to run in-editor.

## Branching
- **main**: stable baseline.
- **feature/*** branches: active development.
- Use pull requests to merge back into **main**.

## Documentation
All documentation now lives in the **Docs/** folder. Start here:
- [Docs/DOCUMENTS_INDEX.md](Docs/DOCUMENTS_INDEX.md)

## Repo Structure (high level)
- **Assets/**: Unity assets (do not reorganize from outside Unity).
- **ProjectSettings/**, **Packages/**, **UserSettings/**: Unity configuration.
- **Docs/**: design docs, catalogs, and system references.
- **Tools/**: scripts/utilities (if added later).
- **.github/**: GitHub workflows/templates.
